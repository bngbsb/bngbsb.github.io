Como criar a sua própria chave para acesso no GitHub?

De acordo com a nova atualização do GitHub, um usuário da plataforma, por questões de segurança, não deve colocar seu nome de usuário e senha para acessar e/ou fazer modificações pelo seu terminal. Agora, é requisitada uma chave de acesso para que ele possa livremente alterar o conteúdo de sua página remotamente. Mas o que é e como criar a sua própria chave de acesso?
Se liga que a resposta está aqui embaixo!!!
  • 1º passo: Verificação
    1. Verifique a existência de uma chave de segurança (SSH).Para isso, abra o terminal e digite ls -al ~/.ssh



    2. Em caso de não aparecerem arquivos chamados "id_rsa.pub", "id_ecdsa.pub" ou "id_ed25519.pub". Se estes nomes não aparecerem, você ainda não tem uma chave SSH pública.
